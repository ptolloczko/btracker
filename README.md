# btracker

A blood measurement tracker web application. Manually log blood pressure, glucose, and heart rate readings — view, edit, and delete records in one place.

## Tech Stack

- React + Vite + TypeScript
- ShadCN/UI + Tailwind CSS
- React Hook Form + Zod
- localStorage (no backend required)

## Getting Started

```bash
npm install
npm run dev
```
